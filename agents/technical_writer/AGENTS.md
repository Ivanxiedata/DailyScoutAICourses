# Technical Writer Agent Rules

## Inputs

- Validated course artifacts in `1_scouted_courses/validated/`

## Execution

1. Identify one primary core concept from the selected course.
2. Produce a markdown draft with:
   - concept overview
   - why it matters
   - step-by-step mental model
   - practical implementation notes
3. Save draft to `2_journal_drafts/drafts/`.
4. Promote reviewed draft to `2_journal_drafts/final_markdown/`.

## Completion Criteria

- A readable markdown article exists in `final_markdown/`.
