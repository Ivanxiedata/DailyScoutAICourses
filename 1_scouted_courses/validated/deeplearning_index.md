# DeepLearning.AI Short Course Index: AI Agents & Generative AI

This index summarizes key learning points from the DeepLearning.AI short course catalog, specifically focusing on AI Agents and Generative AI.

## AI Agents Short Courses

### 1. Multi AI Agent Systems with crewAI
- **Source**: [DeepLearning.AI - Multi AI Agent Systems with crewAI](https://www.deeplearning.ai/short-courses/multi-ai-agent-systems-with-crewai/)
- **Key Learning Points**:
    - **Role-Playing**: Designing agents with specific personas and goals.
    - **Agent Memory**: Implementing short-term, long-term, and shared memory systems.
    - **Tools & APIs**: Connecting agents to external tools (search, APIs) to perform real-world actions.
    - **Collaboration**: Orchestrating multiple agents to work together on complex, multi-step tasks.

### 2. AI Agents in LangGraph
- **Source**: [DeepLearning.AI - AI Agents in LangGraph](https://www.deeplearning.ai/short-courses/ai-agents-in-langgraph/)
- **Key Learning Points**:
    - **Controllable Dataflows**: Building agents with explicit state management using LangGraph.
    - **Persistence**: Managing state across conversation sessions for long-running workflows.
    - **Human-in-the-Loop**: Integrating human feedback and approval steps into agent processes.

### 3. Agentic Design Patterns
- **Source**: [DeepLearning.AI - AI Agents Guide](https://www.deeplearning.ai/the-batch/check-out-our-course-on-how-to-build-ai-agents/)
- **Key Learning Points**:
    - **Reflection**: Agents reviewing and improving their own output.
    - **Planning**: Breaking down complex objectives into sequential sub-tasks.
    - **Tool Use**: Autonomous selection of functions to execute based on context.

## Generative AI Short Courses

### 1. Generative AI for Everyone
- **Source**: [DeepLearning.AI - Generative AI for Everyone](https://www.deeplearning.ai/courses/generative-ai-for-everyone/)
- **Key Learning Points**:
    - **Fundamentals**: How generative models work (capabilities and limitations).
    - **AI Strategy**: Identifying business opportunities and risks associated with GenAI.
    - **Workforce Impact**: Understanding how AI changes modern job roles and workflows.

### 2. ChatGPT Prompt Engineering for Developers
- **Source**: [DeepLearning.AI - ChatGPT Prompt Engineering](https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/)
- **Key Learning Points**:
    - **Iterative Prompt Development**: Refining prompts through a structured development cycle.
    - **Capabilities**: Summarizing, inferring, transforming (translation/formatting), and expanding text.
    - **System Messages**: Using system instructions to control model behavior and safety.

### 3. Retrieval Augmented Generation (RAG)
- **Source**: [DeepLearning.AI - RAG Courses](https://www.deeplearning.ai/courses/)
- **Key Learning Points**:
    - **Knowledge Retrieval**: Efficiently searching private document sets for relevant context.
    - **Context Injection**: Providing retrieved data to the LLM to ground its responses.
    - **Evaluation**: Measuring retrieval accuracy and response faithfulness.
